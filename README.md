# 코딩 테스트 인증 봇 (Coding Test Bot)

JavaScript와 SQLite로 개발된 코딩 테스트 공부를 지속할 수 있도록 돕는 디스코드 인증 봇입니다. 

매일 정해진 시간까지 코딩 테스트 풀이 스크린샷을 인증하고, 미인증 시 벌금을 부과하여 꾸준한 학습 습관을 만들기 위해 개발했습니다. 한국 기준 공휴일과 주말에는 벌금이 없습니다.
